# Life Tracker
React приложение для отслеживания целей и привычек.
Website: https://sllirik.github.io/life-tracker/

## Стек:
- React
- React Router
- useState / useEffect
- LocalStorage
- Vite

## Функционал:
- Добавление задач
- Фильтрация
- Редактирование
- Удаление
- Сохранение в LocalStorage
- Адаптивный интерфейс
